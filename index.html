ConnectWise.Http

Over 1,800 downloads on NuGet!

ConnectWise.Http is a .NET HttpClient wrapper for consuming ConnectWise Manage's REST API. It provides helpers for generating requests for the API as well as utilities for handling deserialization and error handling, and is entirely asynchronous. Since the request builder encapsulates about 95% of the API, excluding the more complex endpoints, it also makes it easier to build your own requests.

When I was working on the networking side of tech at a Managed Service Provider we used ConnectWise's Automate software to monitor all of our clients' networks as well as their endpoints & servers. When we transitioned from our in-house CRM to ConnectWise Manage I saw a development opportunity and wrote a lot of integrations between the two softwares to improve our ticketing & monitoring processes as well as automate some of our billing.

I found no library to help with this so to challenge myself I wrote ConnectWise.Http!

DuoSecurity.Auth.Http

DUO Security is a Two Factor Authentication (2FA) service, and their Auth API is a REST API for adding 2FA functionality to any type of application. DuoSecurity.Auth.Http is another .NET HttpClient wrapper to assist with consuming this API.

This library fully encapsulates the API - handling all deserialization & errors. It is a summation of a lot of things I learned when writing ConnectWise.Http.
